class ChatbotState (*args, **kwargs)

Représente l'état d'un chatbot avec les données pertinentes pour les interactions et calculs associés.

Cette classe modélise les informations nécessaires pour l'interaction d'un chatbot avec l'utilisateur, y compris les questions posées, les documents pertinents trouvés, les calculs optionnels à réaliser, les réponses produites, ainsi que les données liées à l'historique et aux permissions utilisateur.
